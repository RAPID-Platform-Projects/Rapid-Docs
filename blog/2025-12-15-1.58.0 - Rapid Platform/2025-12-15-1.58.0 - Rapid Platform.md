**Release Date:** 15/12/25

# 1.58.0 Rapid Platform

We're excited to announce new improvements live on Rapid Platform **Monday December 15th**!

---

# Improvements

- Workflow timers will now reply instantly, preventing long running process steps from being duplicated
- Inline styles and other HTML sanitisation has been standardised across the platform, you should be able to use a wider range of safe HTML elements in rich text


# Bug Fixes

- Prevented public forms from being able to save their 'saving' state if they fail to finish saving while submitting
- Updated Embedded file component in Public forms to correctly load their embedded file definition in single item scenarios
