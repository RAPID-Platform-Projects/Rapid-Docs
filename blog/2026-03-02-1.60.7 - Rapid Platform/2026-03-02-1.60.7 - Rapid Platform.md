# v1.60.8 — Rapid Platform

We're excited to announce new improvements live on Rapid Platform Monday March 3rd!

## Improvements
* Added city and district selection to the Address component.

## Bug Fixes
* Check that accounts are enabled before inviting users to SharePoint folders.
* Disabled pagination when searching within a data table to prevent infinite network calls.
* Prevented network calls on keypress when using the Address component.
