# v1.62 — Rapid Platform

We're excited to announce new features live on Rapid Platform **Wednesday, April 15th**

### Breaking Change

Dynamically generated views have been restructured. Custom subqueries that rely on internal lookup join aliases will no longer function as expected; update those queries to use the `<table>_join` naming pattern instead of `<table>`.

## Improvements

* You’ll now see a confirmation dialog when deleting an Adaptive Document in Adaptive Designer

## Bug Fixes

* Fixed an issue where the warning icon was displayed on Send Email service tasks despite valid From field configuration
* Fixed an issue where default values exceeded the configured maximum length for Single Line of Text columns
* Removed the Required toggle from Embedded File slots within User Tasks in Workflow
* Fixed an issue where Adaptive Documents appeared on unrelated User Tasks when no documents were configured
* Fixed an issue where Choice fields configured as radio buttons displayed an additional blank option
* Addressed issues with the Open Document dialog in the Adaptive Designer
* Fixed an issue where the `{~~html}` template caused a crash in Fill Document service tasks
