## Welcome

Welcome to the Rapid Developer learning path! It’s great to have you here!

This learning path is for everyone who wants to learn how to build apps with the Mendix Platform. Whether you have a technical background or not, by the end of this learning path, you will be able to use Mendix and create awesome apps.

The Rapid Developer learning path consists of ten modules:

- In module 1, you will be introduced to this learning path, learn about Mendix and where you can find all the information you need.
- In module 2, you will explore how Mendix supports the Agile methodology and enables collaboration between IT and business.
- In module 3, you will use Studio Pro to create the first pages of your app from scratch.
- In module 4, you will see how you can add data to your app, structure it using the Domain Model, and learn about the Team Server.
- In module 5, you will be introduced to associations, which allows connecting different types of data together to form relationships.
- In module 6 you will learn to create microflows, the custom logic of Mendix apps.
- In module 7, you will build more advanced microflows to automate frequent work patterns.
- In module 8, you will learn how to expand your app with more advanced capabilities such as data validation.
- In module 9, you will ensure the security of the data within your app. Problems can arise if users can access information they are not allowed to see. Here you will learn to prevent that.
- In module 10, you will get some practical information about the Rapid Developer Certification as well as some tips on what you can do next.

Each module consists of various smaller units, such as explanations, assignments, and knowledge checks. We recommend you follow all the assignments as they will give you hands-on experience working with Mendix.

_If you get stuck along the way, you can download the solution app in the Resources provided in the Conclusion module._

**Exam**

When you complete this learning path, you can take the exam to receive the Rapid Developer Certificate. The exam consists of 50 multiple-choice questions. To pass the test, your overall score should be above 75%, and your per category scores at least 40% each. Then you will be a certified Mendix Rapid Developer!
